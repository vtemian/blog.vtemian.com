AI agents are powerful but undisciplined. They lose context, skip planning, and produce work without quality gates. I build open-source tools that add **structure**, **memory**, and **visibility** to the agentic development lifecycle.

The common thread: AI shouldn't just write code; it should follow a workflow that humans can trust, share, and learn from.

### Featured Projects

- **[cursouls](/project/cursouls/)**: A pixel cafe in your sidebar. Ambient awareness for Cursor, Claude Code, Codex, and OpenCode agents.
- **[agentprobe](/project/agentprobe/)**: Real-time observability library for AI coding agents. The engine behind Cursouls.
- **[micode](/project/micode/)**: Structured agent workflows (brainstorm/plan/implement) for Claude and OpenCode.
- **[claudebin](/project/claudebin/)**: The "share button" for Claude Code sessions. Publish and embed terminal conversations.
- **[octto](/project/octto/)**: A browser-based UX for agentic exploration, replacing the terminal for complex brainstorming.
- **[claude-notes](/project/claude-notes/)**: convert local Claude Code transcripts into readable, searchable HTML.

Most are [OpenCode](https://opencode.ai) plugins or [Claude Code](https://claude.ai/code) MCP servers. All are MIT licensed.
